# Glossario

Il presente glossario raccoglie in un unico punto tutti i principali termini che sono rilevanti per il progetto,
con l'obiettivo di stabilire e mantenere una tassonomia univoca all'interno del codice.

| Termine             | Significato           |
|---------------------|-----------------------|
| Discount Code       | Buono Sconto          |
| Discount Code Batch | Lotto di Buoni Sconto |
| Discount Code Type  | Tipo di Buono Sconto  |
| Train Service       | Servizio              |
| Train Brand         | Brand (Marchio)       |
| Train Station       | Stazione              |
